# 휴머노이드 격투 경기 관리 웹

라이브 스트리밍 화면에 띄워 **타이머·점수·경기 결과·추첨**을 관리하는 순수 **HTML/CSS/JS** 웹 페이지입니다.  
브라우저만 있으면 어디서나 실행됩니다. (기본 배율 **1.3×**)

---

## ✨ 주요 기능

### 1) 경기 진행 (Referee View)
- 라운드 시간: **1분 / 2분** 선택 (변경 시 리셋 확인)
- 스톱워치: `분:초.1/100초`, 시작/일시정지/리셋
- 효과음 & 화면 깜빡임  
  - **1분 모드:** 30초, 50초, 51~59초 삑 / 60초 긴 삐비비빅  
  - **2분 모드:** 1:00, 1:30, 1:50~1:59 삑 / 2:00 긴 삐비비빅
- 점수 관리(RED/BLUE): `다운 +1`, `슬립다운 +0.5`, `점수 초기화(확인)`, `직전 점수 취소(Undo)`

### 2) 경기 결과
- 진행 시간(초), 파일럿 이름, 점수 **저장 전 확인 팝업**
- 저장 목록 **수정/삭제** 지원(인라인 편집)
- **CSV/TXT** 내보내기  
  - CSV 헤더: `duration_seconds,name_red,score_red,name_blue,score_blue`  
  - 윈도우 엑셀 한글/열 깨짐 방지(**UTF-8 BOM**) 적용

### 3) 투표 추첨
- 인원 수(최대 번호) 설정 → 이름 입력 → **중복 없는 랜덤 번호** 부여
- 결과 **오름차순** 표시
- “번호 뽑기” 시 **말풍선(오버레이)** 로 번호 크게 표시(확인/바깥 클릭/ESC로 닫기)
- **CSV/TXT** 내보내기  
  - CSV 헤더: `name,number` (UTF-8 BOM)

## 🚀 사용 방법

1. 이 폴더에서 **`index.html`** 을 브라우저로 열기(더블클릭 또는 드래그).
2. 상단 탭 이동  
   - **경기 진행:** 타이머·점수 조작 → `경기 결과 저장`  
   - **경기 결과:** 저장 목록 확인/수정/삭제, CSV/TXT 내보내기  
   - **투표 추첨:** 인원 수 설정 → 이름 입력 → `번호 뽑기` → 파일 저장
3. 스트리밍(OBS 등)에서는 **브라우저 소스**로 캡처하고 **전체화면(F11)** 권장.

> 🔊 **사운드 팁**: 브라우저 정책상 **사용자 조작 전에는 오디오가 제한**될 수 있습니다.  
> 페이지 안 버튼을 한 번 눌러 권한을 연 뒤 타이머를 사용하세요.

## 📝 내보내기 포맷 예시

**경기 결과 (CSV)**
```
duration_seconds,name_red,score_red,name_blue,score_blue
83,"Hong","1.5","Lee","2"
```

**투표 추첨 (CSV)**
```
name,number
"Hong","12"
"Lee","3"
```

**TXT**
```
[83s] RED Hong: 1.5 | BLUE Lee: 2
Hong : 12
```
