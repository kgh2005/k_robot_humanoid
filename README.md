# 휴머노이드 격투 경기 관리 웹 페이지

휴머노이드 격투 대회에서 **라이브 스트리밍 화면에 띄워서**  
심판이 타이머·점수·라운드 진행을 관리할 수 있도록 만든 웹 페이지입니다.

HTML / CSS / JavaScript 만으로 동작하며, 브라우저만 있으면 어디서나 사용할 수 있습니다.

---

## 기능 소개

### 타이머
- **1분 / 3분 라운드 모드 선택 가능**
- 실시간 스톱워치 형식 (분:초.1/100초)
- 버튼
  - `시작 / 재개`
  - `일시정지`
  - `리셋` (리셋 전 확인 팝업 표시)
- 라운드 종료 시 자동 정지

### 효과음 & 화면 깜빡임
- **브라우저 Web Audio**를 이용한 삐빅 효과음
- 특정 시점(30초, 50초, 1분 / 1분, 2분, 2분 30초, 2분 50초, 마지막 10초 등)에 삐빅
- 라운드 종료 시 **여러 번 삐빅 → 긴 알림음**
- 알림이 발생할 때마다 메인 패널이 살짝 **깜빡이면서 강조**

### 점수 관리
- RED / BLUE 파일럿 이름 입력 가능
- 점수 버튼
  - `다운 +1`
  - `슬립다운 +0.5`
  - `점수 초기화` (초기화 전 확인 팝업)
- `직전 점수 취소 (Undo)` 버튼으로 바로 전 점수 변경 한 번 되돌리기 가능
